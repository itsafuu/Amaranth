Searching Crypt

Using keywords is the most effective form of searching. For example: DOM obf will show all vampires with superior Dominate and inferior or superior Obfuscate.

cel brujah will show all Brujah vampires will Celerity — alternatively you could use the clan filter and select Brujah.

Once you've filtered on keywords you can then use the filters to select the group or capacity.

Technical Issues

This tool is provided as is; without support and has been developed for modern browsers and devices.

If you're having issues, please ensure your running the latest version of your operating system / mobile browser.

For desktop users, try running the latest version of Chrome or Safari
